# 🏥 AI-Powered Healthcare Chatbot

An AI-powered healthcare chatbot built using **OpenAI API**, **Flask**, and **REST APIs** to 
provide intelligent responses to patient queries. The chatbot assists users with
healthcare-related information and appointment scheduling through natural language conversations. 
It features a Flask backend that securely processes JSON requests and integrates with the OpenAI API. 
Prompt engineering techniques are used to improve response accuracy and deliver relevant, 
context-aware interactions. The project demonstrates the integration of Large Language Models (LLMs) 
intoreal-world healthcare applications using a scalable RESTful architecture.
